# קבלן שלד - אפליקציית עובד מובייל

## 📱 מידע כללי

אפליקציה מובייל עבור עובדי קבלן שלד עם כל הכלים הדרושים לניהול יום עבודה.

## ✨ תכונות

- ✅ **התחלה/סיום עבודה** - כפתור התחלה וסיום עם טיימר בזמן אמת
- ✅ **משימות** - רשימת משימות יומיות עם סטטוס
- ✅ **תמונות** - צילום והעלאה מגלריה
- ✅ **דוחות** - ראה הכנסה ושעות עבודה
- ✅ **התחברות** - התחברות מאובטחת לשרת

## 🚀 בנייה

### GitHub Actions (מומלץ)

```bash
git push origin main
```

GitHub Actions יבנה אוטומטית:
- APK ל-Android
- IPA ל-iOS

### בנייה מקומית

```bash
npm install
npm run android  # או npm run ios
```

## 📥 הורדה

- APK: `/home/ubuntu/KablanWorker-release.apk`
- IPA: בנייה דרך GitHub Actions
